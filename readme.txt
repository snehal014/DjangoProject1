1. Set up your virtual environment in editor
2. Upload all the "MySql Database" details and create a database in mysql for the tables to get stored
which we are creating with the help of models.py file
3. In Terminal write python manage.py makemigrations and python manage.py migrate to easily create tables in database.
4. Then you can run the server using 
   : python manage.py runserver
5. To log in to the page you need to create a superuser using command:
   python manage.py createsuperuser
